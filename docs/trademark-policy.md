# Chronicle Stack 商標・名称利用ポリシー

この文書は、Chronicle Stack の名称、ロゴ、プロジェクト表示の利用方針を説明します。

## 目的

Chronicle Stack はAGPL-3.0-or-laterで公開されており、コードの利用、改変、再配布はライセンス条件に従って可能です。

一方で、Chronicle Stack という名称、ロゴ、公式プロジェクト表示は、利用者が公式版と改変版を混同しないように扱う必要があります。

## 許可される利用

次のような説明的利用は可能です。

- 「Chronicle Stack を利用しています」
- 「Chronicle Stack を基にしたforkです」
- 「Chronicle Stack compatible」
- 技術記事、レビュー、研究、教育目的での言及

## 注意が必要な利用

改変版、派生版、商用サービス、ホスティングサービスで Chronicle Stack の名称を使う場合は、公式版との混同を避ける必要があります。

推奨される表現:

- 「based on Chronicle Stack」
- 「powered by Chronicle Stack」
- 「Chronicle Stack fork」

避けるべき表現:

- 公式版と誤認される名称
- ZYX Corp株式会社が認定、提携、推奨しているように見える表現
- 改変版を単に「Chronicle Stack」として提供する表現

## 禁止される利用

明示的な許可なく、次の利用はできません。

- Chronicle Stackの公式ロゴとして誤認されるロゴ利用
- ZYX Corp株式会社の公式サービスであるかのような表示
- 公式認定、公式提携、公式サポートを示唆する表示
- 利用者に公式版と改変版を混同させる表示

## 商用利用

商用利用そのものを禁止するものではありません。

ただし、Chronicle Stackの名称、ロゴ、公式性を商用サービスで利用する場合、混同を避ける表示が必要です。必要に応じてZYX Corp株式会社との個別契約または商用ライセンスで扱います。

## 連絡

名称、ロゴ、公式表示、商用利用に関する相談は、ZYX Corp株式会社へ問い合わせてください。

## 注意

この文書は商標・名称利用方針の説明であり、法的助言ではありません。正式な判断が必要な場合は、専門家に相談してください。
