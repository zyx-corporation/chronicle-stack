# Federation

このディレクトリは、Chronicle Stack の分散連合、文脈SNS、ノード間共有、撤回、信頼、連合プロトコルに関する文書を置く領域です。

## Documents

- [Federation Requirements](requirements.md): Chronicle Stack を AI時代の文脈主権型SNSとして成立させるための要求仕様。

## Classification

Federation 文書は、ネットワーク同期そのものだけでなく、ローカル正本、最小共有、来歴、RDE監査、撤回、信頼、AI非依存を含む長期要求を扱います。

実装順序やCLI候補は `docs/roadmaps/federation-implementation-roadmap.md` を参照します。
