# Chronicle Stack v1.47.0 Release Notes

Chronicle Stack `v1.47.0` is a local review CLI-equivalent structured-rendering release over the published `v1.46.0` baseline.

## Added

- structured `cli_equivalent_detail` payloads for review action previews and blocked responses
- key-first rendering for review preview summaries and review action detail panes
- regression coverage for structured CLI-equivalent payloads and renderer wiring

## Kept stable

- local-first fail-closed review-write semantics
- append-only Chronicle record authority
- copyable review CLI commands and read-only preview boundaries
