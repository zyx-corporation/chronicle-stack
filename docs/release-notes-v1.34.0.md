# Chronicle Stack v1.34.0 Release Notes

Chronicle Stack `v1.34.0` is a local review-action-preview-summary structured-contract release over the published `v1.33.0` baseline.

## Added

- structured `recovery_summary_key` fields for review-action preview recovery summaries
- structured `follow_up_summary_key` fields for review-action preview follow-up summaries
- key-first rendering for review-action preview summary details

## Kept stable

- local-first read-only preview semantics
- append-only Chronicle record authority
- review CLI command generation and mutation boundaries
