# Chronicle Stack v1.31.0 Release Notes

Chronicle Stack `v1.31.0` is a local review-warning structured-contract release over the published `v1.30.0` baseline.

## Added

- structured `message_key` fields for review warning details
- structured `label_key` and `message_key` fields for overview warning summaries
- structured `summary_key` fields for warning-summary count wording

## Kept stable

- local-first review capability semantics
- read-only UI boundaries
- append-only Chronicle record authority
