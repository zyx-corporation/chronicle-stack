# Chronicle Stack v1.59.0 Release Notes

Chronicle Stack `v1.59.0` is a local invocation/auth/mutation boolean-detail i18n release over the published `v1.58.0` baseline.

## Added

- key-first localized invocation-plan boolean detail lines
- key-first localized auth-boundary and UI-boundary boolean detail lines
- key-first localized mutation-readiness enablement-ready detail lines
- regression coverage for localized boolean detail rendering across adjacent local UI surfaces

## Kept stable

- local-first read-only preview-contract behavior
- append-only Chronicle record authority
- existing invocation-plan, auth-boundary, and mutation-readiness semantics
