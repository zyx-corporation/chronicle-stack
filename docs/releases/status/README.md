# Release Status Docs

このディレクトリは、各 release の状態と対象スコープを記録する status 文書をまとめる領域です。

## Current Pointer

- latest status: `release-status-v1.138.0.md`
- paired readiness: `../readiness/release-readiness-v1.133.md`
- paired smoke: `../smoke/smoke-test-v1.133.md`
- paired remaining: `../remaining/v1.133-release-remaining-issues.md`

## Naming

- file pattern: `release-status-v<version>.md`
- status 文書では lane / status / scope / boundary を短く固定的に残す
- 実行手順や operator 導線は `../operations/` に寄せる
