# Release Remaining Issues Docs

このディレクトリは、各 release 完了後に残る follow-on と未着手課題を記録する文書をまとめる領域です。

## Current Pointer

- latest remaining: `v1.133-release-remaining-issues.md`
- paired notes: `../notes/release-notes-v1.133.0.md`
- paired status: `../status/release-status-v1.133.0.md`
- paired readiness: `../readiness/release-readiness-v1.133.md`

## Naming

- file pattern: `v<version>-release-remaining-issues.md`
- remaining 文書では immediate follow-ons を境界付きで残す
- 実行済み release 手順や template は `../operations/` に重複させない
