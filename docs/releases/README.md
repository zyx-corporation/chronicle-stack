# Releases

このディレクトリは、Chronicle Stack のリリース関連文書を置く領域です。

## Classification

- `notes/`: release notes。
- `readiness/`: release readiness criteria / preparation notes。
- `status/`: release status reports。
- `smoke/`: smoke test profiles。
- `remaining/`: post-release follow-ons / remaining issues。
- `operations/`: release execution, deployment, lock, operator, and tag-policy docs。

## Subdirectory Indexes

- `notes/README.md`
- `status/README.md`
- `readiness/README.md`
- `smoke/README.md`
- `remaining/README.md`
- `operations/README.md`

## Note

リリース種別ごとの文書は `docs/` 直下からこの配下へ整理済みです。新しいリリース文書を追加する場合も、同じ分類に合わせて配置します。
