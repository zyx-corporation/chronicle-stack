# Releases

このディレクトリは、Chronicle Stack のリリース関連文書を置く領域です。

## Planned classification

- `notes/`: release notes。
- `readiness/`: release readiness criteria / preparation notes。
- `status/`: release status reports。
- `policy/`: release policy, tag policy, compatibility policy。

## Note

現時点では、仕様・ロードマップ・分散連合文書の分類を先に整理しています。大量の `release-status-v*.md` は差分が大きくなるため、後続PRで `docs/releases/status/` へまとめて移動する方針です。
