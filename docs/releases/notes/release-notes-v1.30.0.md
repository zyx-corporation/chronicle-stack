# Chronicle Stack v1.30.0 Release Notes

Chronicle Stack `v1.30.0` is a local mutation-summary structured-contract release over the published `v1.29.0` baseline.

## Added

- structured `message_key` fields for list-row mutation readiness summaries
- structured `scope_note_key` fields for mutation summary boundary wording
- structured `remaining_summary_key` fields for first unsatisfied prerequisite summaries
- structured operational-readiness message keys for mutation detail rendering

## Kept stable

- read-only UI boundaries
- local-first mutation gating semantics
- append-only Chronicle record authority
