# Chronicle Stack v1.58.0 Release Notes

Chronicle Stack `v1.58.0` is a local response-metadata and runtime-config detail i18n release over the published `v1.57.0` baseline.

## Added

- key-first localized provider-response finish-reason and provider-status detail lines
- key-first localized runtime-config source, provider-kind, and boolean detail lines
- key-first localized runtime-boundary boolean detail lines
- regression coverage for localized response-metadata and runtime-config detail rendering

## Kept stable

- local-first read-only preview-contract behavior
- append-only Chronicle record authority
- existing runtime-config, runtime-boundary, and provider-response semantics
