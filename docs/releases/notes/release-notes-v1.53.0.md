# Chronicle Stack v1.53.0 Release Notes

Chronicle Stack `v1.53.0` is a local identity-proof compact-badge i18n release over the published `v1.52.0` baseline.

- key-first localized `proof-status=` badges in compact preview and mutation summaries
- structured `required_identity_field_details` reuse for localized `proof-fields=` badges
- regression coverage for localized compact identity-proof summaries

## Kept stable

- local-first read-only preview-contract behavior
- append-only Chronicle record authority
- existing review boundary and mutation readiness semantics
