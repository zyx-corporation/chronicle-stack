# Chronicle Stack v1.52.0 Release Notes

Chronicle Stack `v1.52.0` is a local preview-contract error-summary localization release over the published `v1.51.0` baseline.

## Added

- key-first localized `errors=` badges in compact preview-contract summaries
- structured possible-error detail reuse inside preview-contract rendering
- regression coverage for compact error-summary localization

## Kept stable

- local-first read-only preview-contract behavior
- append-only Chronicle record authority
- existing possible-error payload contracts and copyable CLI actions
