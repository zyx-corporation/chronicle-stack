# Chronicle Stack v1.56.0 Release Notes

Chronicle Stack `v1.56.0` is a local write-route detail-status i18n release over the published `v1.55.0` baseline.

## Added

- key-first localized write-route detail status lines for authorization, assurance, target-state, and identity-proof status
- structured reuse of existing write-route status and field detail contracts in the detail panel
- regression coverage for localized detail status rendering

## Kept stable

- local-first read-only preview-contract behavior
- append-only Chronicle record authority
- existing review route and fail-closed boundary semantics
