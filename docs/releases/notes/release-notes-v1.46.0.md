# Chronicle Stack v1.46.0 Release Notes

Chronicle Stack `v1.46.0` is a local invocation-plan command-detail rendering release over the published `v1.45.0` baseline.

## Added

- structured downstream-command details for invocation-plan payloads
- key-first rendering for invocation-plan downstream command lines
- regression coverage for invocation-plan command-detail payloads and renderers

## Kept stable

- local-first explicit/manual runtime invocation boundaries
- append-only Chronicle record authority
- read-only invocation-plan notice behavior and copyable CLI follow-up commands
