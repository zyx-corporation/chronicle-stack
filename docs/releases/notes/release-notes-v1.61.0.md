# Chronicle Stack v1.61.0 Release Notes

Chronicle Stack `v1.61.0` is a local interactive-UI mutation-token hardening release over the published `v1.60.0` baseline.

## Added

- per-session local mutation token enforcement for browser-triggered review write routes
- route-contract metadata for mutation-token requirements on interactive review flows
- roadmap documentation for the next interactive UI and GraphRAG phases

## Kept stable

- `.chronicle/chronicle.jsonl` primary-record authority
- loopback-local explicit UI boundary
- fail-closed review decision and audit persistence semantics
