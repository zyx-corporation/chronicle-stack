# Chronicle Stack v1.35.0 Release Notes

Chronicle Stack `v1.35.0` is a local review-write-route-failure-family structured-contract release over the published `v1.34.0` baseline.

## Added

- structured `summary_key` fields for review write-route failure families
- structured `summary_key` fields for nested review-action failure-family contracts
- key-first rendering for failure-family contract summaries

## Kept stable

- local-first fail-closed review-write semantics
- append-only Chronicle record authority
- review route status-code and error-family contracts
