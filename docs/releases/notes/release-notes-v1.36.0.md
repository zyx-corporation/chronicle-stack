# Chronicle Stack v1.36.0 Release Notes

Chronicle Stack `v1.36.0` is a local review-write-route-status-code structured-contract release over the published `v1.35.0` baseline.

- structured `when_key` fields for review write-route status-code contract entries

- tests for structured status-code reason contracts

## Kept stable

- local-first fail-closed review-write semantics
- append-only Chronicle record authority
- review route status codes, families, and mutation boundaries
