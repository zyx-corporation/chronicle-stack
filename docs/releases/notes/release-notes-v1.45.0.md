# Chronicle Stack v1.45.0 Release Notes

Chronicle Stack `v1.45.0` is a local review-command-detail structured-rendering release over the published `v1.44.0` baseline.

## Added

- structured recovery-command and follow-up-command details
- key-first rendering for command detail lines on review action panels
- regression coverage for command-detail payloads and renderers

## Kept stable

- local-first fail-closed review-write semantics
- append-only Chronicle record authority
- structured route, possible-error, status-code, authorization, target-state, and failure-family contracts
