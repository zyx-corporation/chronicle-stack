# Chronicle Stack v1.44.0 Release Notes

Chronicle Stack `v1.44.0` is a local review-route-summary structured-detail release over the published `v1.43.0` baseline.

## Added

- structured path and CLI summary keys for review write-route entries
- key-first rendering for action-route and CLI-equivalent detail lines
- regression coverage for route summary payloads and renderers

## Kept stable

- local-first fail-closed review-write semantics
- append-only Chronicle record authority
- structured possible-error, status-code, authorization, target-state, and failure-family contracts
